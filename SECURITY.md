# Security Policy

## Supported Versions

| Component | Supported |
|---|---|
| CKP specification `v0.3.x` | Yes |
| `@clawkernel/sdk` `0.3.x` | Yes |
| `@clawkernel/ckp-test` `0.3.x` | Yes |

## Reporting a Vulnerability

Please report vulnerabilities privately by emailing:

- `security@datastrat.co`

Include:

1. Affected component and version.
2. Reproduction steps or proof of concept.
3. Impact assessment.
4. Any proposed mitigation.

We aim to acknowledge reports within **72 hours** and provide an initial triage within **7 days**.

## Disclosure Process

1. We validate and reproduce the issue.
2. We prepare a fix and tests.
3. We coordinate disclosure timing with the reporter.
4. We publish a security advisory and release notes.

Please do not open public issues for unpatched security vulnerabilities.
